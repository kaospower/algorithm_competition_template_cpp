# 1.0x3f
0x3f常用于给静态数组初始化成无穷大,这是由于memset按照字节赋值,使用0x3f恰好可以与其对应  
0x3f3f3f3f常用于表示INF,其值大约为1e9,使用这个值是因为其不会炸int

# 5.C++取模
```c++
nums[((i+nums[i])%n+n)%n]
```
c++取模和python不同,余数符号和被除数相同,然后对被除数取绝对值后取模,由于第一次模n后可能为负数,因此需要+n之后进行第二次取模
